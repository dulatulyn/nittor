# Nittor

Movie review and recommendation platform. Browse movies, watch trailers, leave reviews, and get recommendations based on what you've watched

## What it does

Users can sign up and log in, browse movies with search and filters, view details like description, rating, and trailer, write and edit reviews (1–5 stars), and receive movie recommendations based on what they’ve watched.

## Tech

- Frontend: Angular
- Backend: Django
- Auth: JWT
- db: SQLite

## How to run

Backend:
```
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Frontend:
```
cd frontend
npm install
ng serve
```
